# Setup

- npm install
- cp .env.sample .env
- npm run start
- open http://localhost:3000
- up backend app [estate_agency_frontend](https://github.com/niten2/estate_agency_frontend)

# Tests

##### fixtures react-cosmos
  - npm run cosmos
  - open http://localhost:8989

##### tests jest
  - npm run tests

# Deploy gh-pages
  - npm run deploy
